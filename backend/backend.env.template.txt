## Backend environment template
## Create `backend/.env` locally with these values.
## (This repo may block creating `.env` via automated tooling; create it manually in your editor.)

# Azure Blob Storage (required)
AZURE_STORAGE_ACCOUNT_NAME=your_account_name
AZURE_STORAGE_ACCOUNT_KEY=your_account_key

# Containers (optional)
AZURE_STORAGE_CONTAINER_INCOMING=incoming-docs
AZURE_STORAGE_CONTAINER_VERIFIED=verified-docs

# API security (must match frontend)
API_TOKEN=veriquickx-secret-token-change-in-production
